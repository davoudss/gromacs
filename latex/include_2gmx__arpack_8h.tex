\subsection{\-Detailed \-Description}
\-Selected routines from \-A\-R\-P\-A\-C\-K. \-This file contains a subset of \-A\-R\-P\-A\-C\-K functions to perform diagonalization and \-S\-V\-D for sparse matrices in \-Gromacs.

\-Consult the main \-A\-R\-P\-A\-C\-K site for detailed documentation\-: \href{http://www.caam.rice.edu/software/ARPACK/}{\tt http\-://www.\-caam.\-rice.\-edu/software/\-A\-R\-P\-A\-C\-K/}

\-Below, we just list the options and any specific differences from \-A\-R\-P\-A\-C\-K. \-The code is essentially the same, but the routines have been made thread-\/safe by using extra workspace arrays.

\-Main \-A\-P\-I of the trajectory analysis library. \-Contains the \-A\-P\-I for the core analysis library.

\-Other headers (centerofmass.\-h, poscalc.\-h) are used internally by the library to calculate positions. \-Analysis tools should preferably not use the routines in these headers directly, but instead get all positions through selections. \-This makes them more flexible with a minimal amount of work.

\subsection{\-Detailed \-Description}
\-Header definitions for the standard \-B\-L\-A\-S library. \-This is the subset of \-B\-L\-A\-S routines used for the linear algebra operations in \-Gromacs. \-Do \-N\-O\-T use this for other purposes -\/ we only provide this as a simple fallback/reference implementation when no optimized \-B\-L\-A\-S is present. \-If you need an implementation for your own code there are several much faster versions out there.

\-All routines are compatible with the \-B\-L\-A\-S reference implementation, meaning they assume fortran-\/style matrix row/column organization.

\-There is plenty of documentation for these routines available at \href{http://www.netlib.org/blas}{\tt http\-://www.\-netlib.\-org/blas} , so there is no point in repeating it here.

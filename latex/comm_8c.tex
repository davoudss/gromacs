\-Split up a group into same-\/colored sub-\/groups ordered by key.

\-This is the main comm creation function\-: it's a collective call that takes a color and a key from each process, and arranges all threads that call t\-M\-P\-I\-\_\-\-Split() withe the same color together into a comm.

\-The rank in the new group will be based on the value given in key.

\-Passing \-T\-M\-P\-I\-\_\-\-U\-N\-D\-E\-F\-I\-N\-E\-D as a color will result in the thread not being part of any group, and getting \-T\-M\-P\-I\-\_\-\-C\-O\-M\-M\-\_\-\-N\-U\-L\-L back in newcomm.

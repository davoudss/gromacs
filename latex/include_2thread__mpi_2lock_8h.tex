\-Fast (possibly busy-\/wait-\/based) lock type

\-This lock type forms an intermediate between the spinlocks and mutexes\-: it is based on a busy-\/wait loop, but yields to the scheduler if the lock is locked. \-This is therefore the preferred type of lock for when waits are expected to be reasonably short.

\-Variables of this type should be initialized by calling \hyperlink{include_2thread__mpi_2lock_8h_ae3ae3143c4eb72a051896065d5aadd74}{t\-M\-P\-I\-\_\-\-Lock\-\_\-init()}.

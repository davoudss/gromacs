\subsection{\-Detailed \-Description}
\-A\-P\-I for handling parameters used in selections. \-There should be no need to use the data structures or call the functions in this file directly unless implementing a custom selection method.

\-More details can be found on the page discussing \hyperlink{selmethods}{custom selection methods}.

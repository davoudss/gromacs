\subsection{\-Detailed \-Description}
\-A\-P\-I for structured and optimized calculation of positions. \-The functions in this header are used internally by the analysis library to calculate positions. \-They can also be used in user code, but in most cases there should be no need. \-Instead, one should write an analysis tool such that it gets all positions through selections.

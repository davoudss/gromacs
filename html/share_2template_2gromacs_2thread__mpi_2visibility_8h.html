Detailed Description

Visibility macros.

These macros enable dynamic library visibility support. Either set the 'TMPI_USE_VISIBILITY', or set the TMPI_EXPORT to the right export statement, when incorporating thread_mpi into a larger project.

All exported functions and classes will be tagged by the visibility macro.

See also:
http://gcc.gnu.org/wiki/Visibility

Detailed Description

Convenience header file for non-MPI compatibility.

This file includes the tMPI header file thread_mpi/tmpi.h, as well as thread_mpi/threads.h and thread_mpi/atomic.h header files. If you'd like to use the components individually, include the relevant header files directly.

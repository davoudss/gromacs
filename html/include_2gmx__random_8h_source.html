00255 /* Return a new gaussian random number with expectation value
00256  * 0.0 and standard deviation 1.0. This routine uses a table
00257  * lookup for maximum speed.
00258  *
00259  * WARNING: The lookup table is 16k by default, which means
00260  *          the granularity of the random numbers is coarser
00261  *      than what you get from gmx_rng_gauss_real().
00262  *          In most cases this is no problem whatsoever,
00263  *          and it is particularly true for BD/SD integration.
00264  *      Note that you will NEVER get any really extreme
00265  *          numbers: the maximum absolute value returned is
00266  *          4.0255485.
00267  *
00268  * threadsafe: yes
00269  */

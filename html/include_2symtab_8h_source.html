00050 /*
00051  * This module handles symbol table manipulation. All text strings
00052  * needed by an application are allocated only once. All references
00053  * to these text strings use handles returned from the put_symtab()
00054  * routine. These handles can easily be converted to address independent
00055  * values by invoking lookup_symtab(). So when writing structures to
00056  * a file which contains text strings, this value can be written in stead
00057  * of the text string or its address. This value can easily be converted
00058  * back to a text string handle by get_symtab_handle().
00059  */

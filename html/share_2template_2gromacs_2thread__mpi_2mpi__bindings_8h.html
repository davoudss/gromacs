Detailed Description

MPI bindings for thread_mpi/tmpi.h.

This file contains only macros and redefinitions to expose the standard MPI API with thread_mpi calls.

This is different from the API exposed through thread_mpi/tmpi.h, which uses names like tMPI_Send() instead of MPI_Send()

See also:
thread_mpi/tmpi.h for documentation of the available data types and functions
http://www.mpi-forum.org/docs/docs.html for MPI documentation.

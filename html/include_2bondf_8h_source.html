00079  *     specifies which atoms are involved in a bond of a certain
00080  *     type, see also struct t_idef.
00081  *   t_functype *functype
00082  *      defines for every bonded force type what type of function to
00083  *     use, see also struct t_idef.
00084  *   t_iparams *forceparams
00085  *      defines the parameters for every bond type, see also struct
00086  *     t_idef.
00087  *   real epot[NR_F]
00088  *     total potential energy split up over the function types.
00089  *   int *ddgatindex
00090  *     global atom number indices, should be NULL when not using DD.
00091  *   gmx_bool bPrintSepPot
00092  *     if TRUE print local potential and dVdlambda for each bonded type.
00093  *   int step
00094  *     used with bPrintSepPot
00095  *   return value:

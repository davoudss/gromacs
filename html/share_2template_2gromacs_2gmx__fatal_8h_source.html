00091 /*
00092  * Routine gmx_fatal prints
00093  *
00094  *  "fatal error file %s line %s \n\t "
00095  *
00096  * followed by the string specified by fmt and supplied parameters. If
00097  * errno is 0, only the message and arguments are printed. If errno is
00098  * a legal system errno or -1, a perror like message is printed after the
00099  * first message, if errno is -1, the last system errno will be used.
00100  * The format of fmt is that like printf etc, only %d, %x, %c, %f, %g and %s
00101  * are allowed as format specifiers.
00102  *
00103  * In case all MPI processes want to stop with the same fatal error,
00104  * use gmx_fatal_collective, declared in gmx_fatal_collective.h,
00105  * to avoid having as many error messages as processes.
00106  *
00107  * Tip of the week:
00108  * call this function using the FARGS macro:
00109  * gmx_fatal(FARGS,fmt,...)
00110  *
00111  */

00292 /*
00293  * The struct t_ilist defines a list of atoms with their interactions.
00294  * General field description:
00295  *   int nr
00296  *  the size (nr elements) of the interactions array (iatoms[]).
00297  *   t_iatom *iatoms
00298  *  specifies which atoms are involved in an interaction of a certain
00299  *       type. The layout of this array is as follows:
00300  *
00301  *    +-----+---+---+---+-----+---+---+-----+---+---+---+-----+---+---+...
00302  *    |type1|at1|at2|at3|type2|at1|at2|type1|at1|at2|at3|type3|at1|at2|
00303  *    +-----+---+---+---+-----+---+---+-----+---+---+---+-----+---+---+...
00304  *
00305  *  So for interaction type type1 3 atoms are needed, and for type2 and
00306  *      type3 only 2. The type identifier is used to select the function to
00307  *  calculate the interaction and its actual parameters. This type
00308  *  identifier is an index in a params[] and functype[] array.
00309  */

00355 /*
00356  * The struct t_idef defines all the interactions for the complete
00357  * simulation. The structure is setup in such a way that the multinode
00358  * version of the program  can use it as easy as the single node version.
00359  * General field description:
00360  *   int ntypes
00361  *  defines the number of elements in functype[] and param[].
00362  *   int nodeid
00363  *      the node id (if parallel machines)
00364  *   int atnr
00365  *      the number of atomtypes
00366  *   t_functype *functype
00367  *  array of length ntypes, defines for every force type what type of
00368  *      function to use. Every "bond" with the same function but different
00369  *  force parameters is a different force type. The type identifier in the
00370  *  forceatoms[] array is an index in this array.
00371  *   t_iparams *iparams
00372  *  array of length ntypes, defines the parameters for every interaction
00373  *      type. The type identifier in the actual interaction list
00374  *      (ilist[ftype].iatoms[]) is an index in this array.
00375  *   gmx_cmap_t cmap_grid
00376  *      the grid for the dihedral pair correction maps.
00377  *   t_iparams *iparams_posres
00378  *  defines the parameters for position restraints only.
00379  *      Position restraints are the only interactions that have different
00380  *      parameters (reference positions) for different molecules
00381  *      of the same type. ilist[F_POSRES].iatoms[] is an index in this array.
00382  *   t_ilist il[F_NRE]
00383  *      The list of interactions for each type. Note that some,
00384  *      such as LJ and COUL will have 0 entries.
00385  */

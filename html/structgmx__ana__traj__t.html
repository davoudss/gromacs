gmx_ana_selection_t** gmx_ana_traj_t::sel

Array of selection information (one element for each index group).

After the call to gmx_ana_init_selections(), this array contains information about the selections the user has provided. The array contains ngrps elements; if nanagrps was -1, the number may vary. See gmx_ana_selection_t for details of the contents.

The largest possible index groups for dynamic selections can be found in sel[i]->g, i.e., the program can assume that any index group passed to gmx_analysisfunc() is a subset of the provided group. After gmx_ana_do(), the same groups can be used in post-processing.
